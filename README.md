# 🚆 Train Consist Management System

A simple console-based Java application to manage train coach arrangements using core data structures.

## 📌 Features
- Add and remove coaches
- Reorder coach sequence
- Prevent duplicate entries
- Display current train consist

## ⚙️ Tech Used
- Java
- Data Structures (Linked List, etc.)
